## 🚀 Quick Start

### Play Online
Just visit [harshulmoon.github.io](https://harshulmoon.github.io) — no setup required!
If the main site is blocked, use any other URL on the **Server** tab;
(https://harshulgoon.github.io), (https://harshulmoon.pages.dev)

### Install as Desktop App
1. Visit the site in **Chrome** or **Edge**
2. Click the **📥 Download App** button
3. Click "Install" in the popup
4. The site opens as a desktop app (no browser UI!)

---

## 🎨 Customization

### Change Background
1. Click **🖼️ Change Background**
2. Upload an image (under 5MB)
3. Click **✕ Reset** to restore default

### Toggle Stealth Mode
1. Click **🔒 Cloak Site**
2. Opens in a disguised popup window
3. Perfect for... privacy 👀

### Brand Toggle (Top Right)
Switch between **Compass** and **Dashboard** themes using the toggle switch.
